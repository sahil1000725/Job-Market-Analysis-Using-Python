# Job Market Analysis & Job Scraper Using Python

## 📌 Project Overview

This project is a Python-based Job Market Analysis system that collects job listings from Internshala through web scraping and performs data analysis to identify hiring trends, top recruiting companies, and the most in-demand skills.

The project demonstrates practical applications of Web Scraping, Data Analysis, Data Cleaning, Object-Oriented Programming (OOP), and Data Visualization using Python.

---

## 🎯 Objectives

* Scrape job listings from Internshala.
* Extract important job information.
* Analyze hiring trends.
* Identify the most demanded skills.
* Find top hiring companies.
* Visualize job market insights.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Requests
* BeautifulSoup
* Matplotlib
* Jupyter Notebook

---

## 📊 Features

### Job Data Collection

* Scrapes job listings from Internshala.
* Extracts:

  * Job Title
  * Company Name
  * Required Skills

### Data Processing

* Cleans and structures scraped data.
* Removes inconsistencies and duplicates.

### Market Analysis

* Identifies top hiring companies.
* Finds the most in-demand skills.
* Provides insights into current hiring trends.

### Data Visualization

* Bar charts for top companies.
* Skill demand analysis.
* Graphical representation of job market trends.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Job-Market-Analysis-Using-Python.git
```

Move into the project folder:

```bash
cd Job-Market-Analysis-Using-Python
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:


Open:

